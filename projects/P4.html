TuneUP is a music platform that allows users to upload and play their own songs anytime. Inspired by the Spotify design, TuneUP features a modern design and a user-friendly interface for a seamless music experience.

Technologies: HTML, CSS, JavaScript

The project is still under development, but future plans include enabling TuneUP to be easily hosted using Docker for maximum flexibility.
